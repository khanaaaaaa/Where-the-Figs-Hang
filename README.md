# Where-the-Figs-Hang

Demo: https://khanaaaaaa.github.io/Where-the-Figs-Hang/

# What It Is

An interactive desk scene dedicated to Sylvia Plath. The bookshelf on the right holds four clickable books covering her life, work, legacy, and words, and the typewriter cycles through her quotes.

# Why I Made It

Everyday, I see the fig tree theory analogy like clock work. I adore Sylvia Plath and her works and just wanted to make something that I could use to display some of her works or some facts.

# How I Made It

Tech Stack:

- HTML
- CSS
- JavaScript

The paper poem placement to the typewriter but genuinely a pain to fix but persevered regardless with a lot of trial and error.

# What I Learnt and What I Struggled With

The hardest part was the typewriter paper. Getting a positioned element to sit flush on top of another absolutely positioned element and not get clipped by overflow hidden took a lot of trial and error. I also struggled with the bookshelf, making books look three-dimensional was a struggle. I also learnt the hard way that encoding matters, a Windows append operation corrupted the HTML file with random characters and broke everything.
